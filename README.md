## 近期返澳的同学的建议及症状处置提示

近期抵澳同学请自觉在家**隔离7天**，如果觉得不舒服**请不要去家庭诊所** **也不要去医院** 请直接联系 communicable disease control branch 会有专门的医生来为您检查（免费）。
